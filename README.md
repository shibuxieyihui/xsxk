# 学生选课系统

## 项目概述

本项目是面向高校师生的校园选课管理系统，主要服务于在校大学生与教师。系统实现课程选择、退课、课程管理、权限控制、信息公告等功能，采用前后端分离架构，提升选课流程的效率与便捷性。

## 核心功能

1. 用户身份认证与权限管理：支持学生、教师角色登录与权限隔离。
2. 课程选课与退选管理：学生可在线选课、退课，实时更新课程容量。
3. 课程全生命周期管理：支持课程新增、编辑、发布、下架、归档等操作。
4. 用户资源管理：统一管理师生信息、选课记录、课程归属数据。
5. 信息发布与公告管理：支持发布通知、公告，方便师生获取最新信息。
